Learning Roadmap

We’ll start with these Beginner-Friendly Certifications for Azure.

  • AZ-900: Azure Fundamentals
  • DP-900: Azure Data Fundamentals
  • AI-900: Azure AI Fundamentals

Each certification lists the recommended learning paths for preparation (see below).

1. Azure Fundamentals: AZ-900

LEARNING PATHS

  • Path 1.1: Describe Core Cloud Concepts
  • Path 1.2: Describe Core Azure Services
  • Path 1.3: Describe Core Solutions & Management Tools
  • Path 1.4: Describe General Security & Network Security Features
  • Path 1.5: Describe Identity, Governance, Privacy and Compliance Features
  • Path 1.6: Describe Cost Management & Service Level Agreements
